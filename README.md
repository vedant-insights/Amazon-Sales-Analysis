# Amazon Sales Analysis (EDA Project)

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on an Amazon sales dataset to uncover trends, patterns, and business insights.

## Tools & Technologies

* Python 
* Pandas
* NumPy
* Matplotlib / Seaborn

## Dataset

* Amazon sales dataset containing product, category, price, and sales details.

## Key Analysis Performed

* Data Cleaning & Preprocessing
* Handling Missing Values
* Sales Distribution Analysis
* Category-wise Performance
* Trend Analysis

## Key Insights

* Identified top-performing product categories
* Observed pricing patterns affecting sales
* Found trends in customer purchasing behavior

## How to Run

1. Download the `.ipynb` file
2. Open in Jupyter Notebook
3. Run all cells
